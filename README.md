# 고품격 Todolist제작

## 1. 프로젝트 개요
### 목표
- 사람들의 정보를 저장하는 명함형 계획 프로그램 제작
- 개인 일정 및 연락처 통합 관리 시스템

### 타겟
- 영업직 종사자
- 많은 사람들을 만나는 직군
- 연락처 관리가 필요한 사용자

## 2. 기술 스택
- Next.js
- TypeScript
- TailwindCSS
